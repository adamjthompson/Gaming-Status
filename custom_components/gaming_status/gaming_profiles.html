Configurator

1. User Profiles

Enter Gamer Tags exactly as they appear. They will be formatted for Home Assistant automatically. Blank entries are ignored.

2. Global Exclusions

Apps, dashboards, or video players that should NEVER trigger a "playing" status for anyone. One per line.

3. Title Cleanups

Phrases that will be automatically removed from any game title (e.g., removing "Director's Cut" to help find cover art). One per line.

4. Game Title Overrides

Force the integration to rename a specific game. Excellent for mapping various launcher editions to a single clean name.

5. Custom Cover Art

Provide direct image URLs for games not found on SteamGridDB, or use local files (e.g., /local/gaming_status/art.png). URLs must start with https:// or /local/.
Global Settings (Optional)

These variables control how the integration handles network drops, game crashes, and short sessions. All values are in seconds. Do not change these unless you understand how these will impact the functionality of the integration!

Default: 300. Bridges temporary sensor or network drops.
Default: 600. Time before an Idle/Away status drops to Offline.
Default: 120. Combines sessions during game swaps or crashes.
Default: 300. Sessions shorter than this will be discarded.
