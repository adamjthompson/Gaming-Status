Configurator

1. Notification Methods

 Define your master notification methods here (Discord channels, SMS numbers, Mobile apps). Once created, you can assign these to any player below.

2. User Profiles

 Enter Gamer Tags exactly as they appear. They will be formatted for Home Assistant automatically. Blank fields are ignored.

3. Global Exclusions

 Apps, dashboards, or video players that should NEVER trigger a "playing" status for anyone. One per line.

4. Title Cleanups

 Phrases that will be automatically removed from any game title (e.g., removing "Director's Cut" to help find cover art). One per line.

5. Game Title Overrides

 Force the integration to rename a specific game. Excellent for mapping various launcher editions to a single clean name.

6. Custom Cover Art

 Provide direct image URLs for games not found on SteamGridDB, or use local files (e.g., /local/gaming_status/art.png). URLs must start with https:// or /local/.

7. Weekly Report

 Automatically generate a Markdown report summarizing last week's gaming activity and send it to any of your configured notification services.
